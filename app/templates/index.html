About the Project

This project was created for CS121B (Computer Organization) at the University of Vermont. It leverages multiple technology stacks including the following:

  • Raspberry Pi running a LAMP stack
  • Grove Seeed Studio GSR (Galvanic Skin Response) and MAX30102 Pulse Oximeter Sensors
  • Bootstrapped Flask app providing the web interface to control and visualize data from the sensors

The project allows you to run "Tests" on users where you can ask them multiple questions and associate the questions with the set of readings produced by the sensors while providing live graphs/tables to show the sensor data. It also provides the ability to look back at previous tests and view the patterns of sensor data. Essentially, the Raspberry Pi is able to act as a polygraph machine.

Future Work

Currently, the project is using sensors that are not medical grade. Future work may involve either improving data collection techniques to avoid erroneous readings (particularly using the MAX30102) and improve the ability to distinguish from noise in the readings.

Another potential for further development would involve implementing a binary classification machine learning model to automatically determine whether a question is marked as a Pass/Fail whether than the test administrator doing so.
